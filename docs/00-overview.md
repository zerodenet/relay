# 00 · 产品概览

## 一句话定位

> **以订阅节点池为隧道、对外做透明 L4 端口转发的中转编排面板。**

## 核心价值

- 部署在国内服务器，对外暴露**裸 L4 入口端口**（mixed/socks/http，**无认证**）
- 通过订阅来的境外代理节点构成隧道，将流量转发出墙
- 可选追加落地节点形成链式代理（中转 → 落地）
- **不做任何用户/认证逻辑**：客户协议原样穿过；面板只负责"端口 + 目标 + 隧道链"
- 以**入口端口**为最小计费单元，**双向流量统计**

## 核心抽象：转发规则（Forward）

一条转发规则 = **本地端口 + 监听协议 + (可选)目标站点 + 隧道链**

```
┌──────────────────────────────────────────────────────────────────┐
│                                                                    │
│   入口端口 ──→  隧道链[1..N]  ──→  目标站点 / 客户指定目标            │
│  (mixed/socks/http,                                               │
│   无认证)                                                         │
│      │                                                            │
│      └─ 双向计量「入向 + 出向」字节                                  │
│                                                                    │
└──────────────────────────────────────────────────────────────────┘
```

## 与同类产品对比

| 维度 | 极光面板（gost 转发型）| **Zero Relay Panel** |
|---|---|---|
| 入站层 | L4 端口转发 | **L4-ish（mixed/socks/http 无认证）** |
| 隧道形态 | 手动配中转 IP / SSH 隧道 | **代理协议订阅节点池（自动测速、分组、链式）** |
| 出站层 | L4 转 L4 | **L7 代理协议出站**（订阅节点） |
| 认证 | 无 | **无**（认证由穿过的协议字段自负） |
| 用户系统 | 无 | 无 |
| 计费维度 | 端口/隧道 | **入口端口** |

## 不做的事

- ❌ 用户系统、订阅链接生成、二维码、客户端导入
- ❌ 域名/IP 分流规则集
- ❌ TUN / 系统代理 / GUI 客户端
- ❌ 入口协议层认证（mixed 入站无 user/pass）
- ❌ 多角色权限管理

## 目标用户

- 自建机场，需要把"国内入口"和"出墙隧道"解耦
- 想用订阅节点池替代昂贵 IEPL 专线的中转运营者
- 个人/团队需要"一个端口对应一组境外隧道"的运维场景

## 形态

- **单 Node 进程**（Fastify BFF + 内嵌 Vue SPA 静态资源）
- 部署在 Linux 服务器上，systemd 托管
- 浏览器访问面板，本机或局域网即可

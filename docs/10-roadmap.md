## 不在路线图内（已明确不做）

- ❌ 用户管理 / 计费 / 订阅链接生成（机场层面的事）
- ❌ Web 客户端 / 桌面客户端
- ❌ 域名/IP 分流规则（不属于中转网关定位）
- ❌ TUN / 系统代理
- ❌ 多语言完整支持（仅保留中英）

## 风险与依赖

| 风险 | 影响 | 缓解 |
|---|---|---|
| Zero 内核协议覆盖不足 | 部分订阅节点解析后无法下发 | 编译器内置兼容层 + 不支持节点过滤 + 上游提 PR |
| `config.apply` 整体替换断流 | 频繁配置变更影响业务 | 防抖合并 + 增量下发（待内核支持） |
| 订阅源风控 | 节点失效 | 拟人化 UA + 限流 + 多源冗余 |
| 节点池频繁变化 | 隧道成员抖动 | auto_sync 防抖 + 黑名单退避 |
| 内核版本升级破坏配置 | 升级失败 | dialect 分版本 + 升级前 dry-run + 回滚机制 |
